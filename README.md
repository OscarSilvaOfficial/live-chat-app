# live-chat-app

Frontend de uma aplicação live chat.

Essa aplicação tem como objetivo estudar o VueJs e Socket io
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
